# Starling2-Chromatic-Aberration-Filter
A custom filter for Chromatic Aberration effect, as an extension for Starling2


Usage:
```as3
var filter1:CAFilter = new CAFilter();
filter1.angle = 2;
filter1.intensity = 20;
mySprite.filter = filter1;
```
